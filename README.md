# enable virtual env
install python 3

## macOS

```
$ python3 -m venv env
$ . env /bin/activate 
```

```.vscode/setting.json```

```json
{
    "python.pythonPath": "env/bin/python"
}
```

## Windows
```
> python -m venv env
```

```.vscode/setting.json```

```json
{
    "python.pythonPath": "${workspaceFolder}\\env\\Scripts\\python.exe"
}
```


## pip install
```
(env)$ pip install numpy
```
